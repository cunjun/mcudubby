## Ownership rules

| Content | Canonical location | Notes |
| --- | --- | --- |
| Product overview | Root READMEs | Keep short; link here for detail |
| Setup and workflows | `docs/` | Do not duplicate tool catalogues |
| Tool signatures | `tool-reference.md` and source | Examples must match runtime signatures |
| Release history | `CHANGELOG.md`, `docs/releases/` | Roadmaps move to `docs/archive/` after release |
| Skill references | `skills/mcubug/references/` | Generated by `sync_references.py`; do not edit directly |

Run `python scripts/validate_docs.py` after documentation changes.
