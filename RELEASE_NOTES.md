# mcudubby v0.4.0

Initial public release of `mcudubby`, an MCP server for AI-assisted MCU debugging and real-board firmware diagnosis.

## Highlights

- Probe control and execution tools for halt, resume, reset, stepping, breakpoints, watchpoints, registers, memory, and flash workflows.
- pyOCD and J-Link backend support.
- ELF/DWARF source context, stack inspection, symbol lookup, and source navigation helpers.
- UART, RTT, SVD, RTOS, GDB server, and board bring-up tooling.
- AI-oriented diagnosis flows for startup failures, HardFaults, stack overflow, peripheral issues, and debug loops.
- English and Chinese documentation plus bundled `mcubug` skill references.
